Temperature/Humidity

Tutorial

AWS Server - it is assumed you already have an AWS account as well as Arduino

1: Open and connect to an EC2 instance running on Ubuntu with the following settings: AMI Ubuntu version 20.4 LTS, t.2 micro, vockey as the key pair, with a new security group and 28 GiB

2: Click on the instance ID and go to the security tab

3: Click on the blue security group link and go to edit inbound rules

2: Run the command between the two single quotes:
'curl -s https://gist.githubusercontent.com/tejaswigowda/f289e9bff13d152876e8d4b3281142f8/raw/df37df2e16a3166e686357a045751a7c18bbeebe/installNodeMongoUbuntu16.04.sh | bash'


3: Go to the class repository at: 'https://github.com/tejaswigowda/ame498-598Fall2023/tree/main'

4: Press the blue 'Code' button and copy the URL


5: Return to your EC2 Server, and enter 'git clone ' and then paste the URL, let the command run its course


6: CD to AME498-598Fall2023/ASSIGNMENTS/A4/code/captureDataServer-TempHum


7: type in node server.js to run the server

Arduino

1: Open Arduino


2: Open the DHT11Default.ino file


3: Replace ssid and password to the network/hotspot you have access to


4: Replace the placeholder IP address with your specific EC2 server's public-facing IP address


5: Run the Arduino IDE file
